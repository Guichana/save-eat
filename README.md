## 프로젝트 소개
- 맛있게 먹었던 식사를 기록하고 나중에 다시 찾아볼 수 있는 서비스
- 사진, 가격, 한줄평, 태그 등을 기록해서 이후에 편하게 찾아볼 수 있습니다
<br/>

## 프로젝트 목표
- 모바일에서 사용 가능한 Progressive Web App 개발
- 새로운 개발 환경 학습(Java, spring boot, aws)
